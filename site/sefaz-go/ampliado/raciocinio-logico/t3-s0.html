4. Sequências, matrizes e análise combinatória

PA e PG

As progressões são sequências numéricas que seguem uma regra de formação constante. Na Progressão Aritmética (PA), cada termo resulta da soma do anterior com uma constante chamada razão (r). Na Progressão Geométrica (PG), cada termo resulta da multiplicação do anterior por uma constante chamada razão (q). Reconhecer o padrão e aplicar as fórmulas de termo geral e de soma resolve a maioria das questões de sequências.

Esses conteúdos conectam-se à matemática financeira (juros simples têm crescimento de PA; juros compostos, de PG) e à estatística, sendo por isso muito valorizados em provas de Auditor-Fiscal.

Progressão Aritmética (PA)

Termo geral: an = a1 + (n − 1)·r, onde a1 é o primeiro termo, n a posição e r a razão. A razão é r = an − a(n-1), constante. PA crescente tem r > 0; decrescente, r < 0; constante, r = 0.

Soma dos n primeiros termos: Sn = (a1 + an)·n / 2. Propriedade útil: em uma PA, cada termo é a média aritmética dos seus vizinhos, isto é, ak = (a(k-1) + a(k+1)) / 2.

Progressão Geométrica (PG)

Termo geral: an = a1·q^(n − 1), onde q é a razão, com q = an / a(n-1), constante. Soma dos n primeiros termos (q ≠ 1): Sn = a1·(q^n − 1) / (q − 1).

Propriedade: cada termo é a média geométrica dos vizinhos, ak² = a(k-1)·a(k+1). Para PG infinita com |q| < 1, a soma converge: S = a1 / (1 − q), fórmula útil em dízimas periódicas e em valor presente perpétuo.

Identificando o padrão

Se a DIFERENÇA entre termos consecutivos é constante, é PA (ex.: 3, 7, 11, 15 → r = 4). Se o QUOCIENTE entre termos consecutivos é constante, é PG (ex.: 2, 6, 18, 54 → q = 3).

Há ainda sequências mistas, recursivas (Fibonacci: cada termo é a soma dos dois anteriores) e figurais (padrões geométricos). Nessas, a estratégia é descobrir a lei de formação observando as diferenças de primeira e segunda ordem.

Passo a passo de resolução

1) Identifique se é PA ou PG calculando diferença e quociente. 2) Extraia a1 e a razão. 3) Use o termo geral para achar o termo pedido ou n. 4) Use a fórmula de soma quando pedirem o total.

Dica de cálculo: para achar n a partir de an em PG, use logaritmos: de an = a1·q^(n-1), isole q^(n-1) e aplique log. Em PA, isole n diretamente da equação linear.

💡 Exemplo aplicado

Exemplo PA: numa PA, a1 = 5 e r = 3. Qual o 10º termo e a soma dos 10 primeiros? an = 5 + (10−1)·3 = 5 + 27 = 32. Logo a10 = 32. S10 = (5 + 32)·10 / 2 = 37·5 = 185.

Exemplo PG: numa PG, a1 = 2 e q = 3. Qual o 5º termo e a soma dos 5 primeiros? a5 = 2·3^(5−1) = 2·81 = 162. S5 = 2·(3^5 − 1) / (3 − 1) = 2·(243 − 1)/2 = 242. Logo a soma é 242.

Exemplo de PG infinita (dízima): 0,9999... = 9/10 + 9/100 + ... com a1 = 9/10 e q = 1/10. S = (9/10)/(1 − 1/10) = (9/10)/(9/10) = 1.

🎯 Na prova

Atenção ao expoente (n−1) no termo geral — esquecê-lo e usar n é o erro mais frequente. Na soma da PG, confira o sinal de (q−1): se q<1 use a forma a1·(1−q^n)/(1−q) para evitar números negativos. A banca costuma misturar a contagem da posição (a 'primeira' parcela é n=1, não n=0). Em sequências figurais, calcule as diferenças de segunda ordem quando a de primeira ordem não for constante.

Pontos-chave

  • PA — termo geral: an = a1 + (n−1)·r.
  • PA — soma: Sn = (a1 + an)·n / 2.
  • PG — termo geral: an = a1·q^(n−1).
  • PG — soma finita: Sn = a1·(q^n − 1)/(q − 1).
  • PG infinita com |q|<1: S = a1/(1 − q).
  • PA: diferença constante; PG: quociente constante.
  • Juros simples seguem PA; juros compostos seguem PG.
