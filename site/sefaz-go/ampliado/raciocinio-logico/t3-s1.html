4. Sequências, matrizes e análise combinatória

Análise combinatória

A análise combinatória estuda técnicas de contagem do número de maneiras de organizar ou escolher elementos de um conjunto. O alicerce é o Princípio Fundamental da Contagem (princípio multiplicativo): se uma decisão pode ser tomada de m modos e outra de n modos, então as duas em sequência podem ser tomadas de m·n modos.

A partir desse princípio derivam-se as fórmulas de permutação, arranjo e combinação. A distinção essencial — e a mais cobrada — é se a ORDEM dos elementos importa (arranjo/permutação) ou não importa (combinação). Esse conteúdo é base para probabilidade e aparece em questões de senhas, comissões, filas e sorteios.

Princípio multiplicativo e fatorial

Princípio multiplicativo: multiplicam-se as quantidades de opções de cada etapa independente. Ex.: 3 camisas e 4 calças geram 3·4 = 12 combinações de roupa.

Fatorial: n! = n·(n−1)·(n−2)·...·1, com 0! = 1 e 1! = 1. O fatorial conta de quantas formas n elementos distintos podem ser ordenados. Ex.: 5! = 120.

Permutação

Permutação simples de n elementos distintos: Pn = n!. É o número de ordenações possíveis de todos os elementos. Ex.: anagramas de uma palavra com letras distintas.

Permutação com repetição: quando há elementos iguais, divide-se pelo fatorial das repetições. Para a palavra com n letras, das quais a se repetem a vezes e b se repetem b vezes: P = n! / (a!·b!). Ex.: anagramas de 'ARARA' = 5!/(3!·2!) = 120/12 = 10.

Arranjo (ordem importa)

Arranjo simples de n elementos tomados p a p: A(n,p) = n! / (n − p)!. Conta sequências ordenadas de p elementos escolhidos entre n. A ordem importa: trocar a posição gera arranjo diferente.

Exemplo: quantos números de 2 algarismos distintos com os dígitos 1, 2, 3? A(3,2) = 3!/(3−2)! = 6/1 = 6. São: 12, 13, 21, 23, 31, 32.

Combinação (ordem não importa)

Combinação simples de n elementos tomados p a p: C(n,p) = n! / [p!·(n − p)!]. Conta subconjuntos de p elementos, sem se importar com a ordem. Trocar a ordem NÃO gera combinação nova.

Exemplo: de quantas formas formar uma comissão de 2 pessoas entre 4? C(4,2) = 4!/(2!·2!) = 24/4 = 6. Relação útil: A(n,p) = C(n,p)·p!, ou seja, arranjo é combinação multiplicada pelas ordenações.

💡 Exemplo aplicado

Questão: uma comissão da SEFAZ-GO terá 1 presidente, 1 vice e 1 secretário, escolhidos entre 6 auditores. De quantas formas? Como os cargos são distintos, a ORDEM importa: usamos arranjo. A(6,3) = 6!/(6−3)! = 6!/3! = (6·5·4·3!)/3! = 6·5·4 = 120. Resposta: 120 maneiras.

Variação que muda tudo: se a comissão fosse de 3 membros SEM cargos definidos, a ordem não importa: C(6,3) = 6!/(3!·3!) = 720/(6·6) = 20. Note a diferença: 120 (com cargos) versus 20 (sem cargos). Identificar 'ordem importa?' é o que separa as duas respostas.

🎯 Na prova

A pergunta-chave é sempre 'a ordem dos elementos altera o resultado?'. Cargos distintos, pódios, senhas e filas → ARRANJO. Comissões, grupos, escolhas de itens 'iguais em status' → COMBINAÇÃO. A pegadinha favorita é descrever uma comissão com cargos (arranjo) usando linguagem de 'escolher pessoas' para induzir você a usar combinação. Cuidado também com restrições (elementos que devem ficar juntos ou separados): resolva por etapas com o princípio multiplicativo.

Pontos-chave

  • Princípio multiplicativo: multiplique as opções de cada etapa.
  • Ordem importa → arranjo/permutação; não importa → combinação.
  • Permutação: Pn = n!; com repetição, divide pelos fatoriais repetidos.
  • Arranjo: A(n,p) = n!/(n−p)!.
  • Combinação: C(n,p) = n!/[p!·(n−p)!].
  • Relação: A(n,p) = C(n,p)·p!.
  • Convenção: 0! = 1.
