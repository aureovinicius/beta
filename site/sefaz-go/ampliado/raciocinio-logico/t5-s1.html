6. Matemática Financeira

Descontos

Desconto é o abatimento concedido sobre o valor de um título (como uma duplicata ou nota promissória) quando este é pago ou negociado antes do vencimento. O valor escrito no título, devido na data de vencimento, é o valor nominal (N) ou valor de face; o valor pago antecipadamente é o valor atual (A) ou valor líquido. O desconto (D) é a diferença: D = N − A.

Há dois regimes de cálculo do desconto simples — comercial (ou bancário) e racional (ou 'por dentro') — que diferem quanto à base de incidência da taxa. Distingui-los é o ponto central das questões de prova.

Desconto comercial (por fora / bancário)

A taxa de desconto incide sobre o VALOR NOMINAL (de face). Fórmula: Dc = N·i·t. O valor atual fica: Ac = N − Dc = N·(1 − i·t).

É o mais usado por bancos em operações de antecipação de recebíveis, pois favorece o credor (gera desconto maior). Por isso também é chamado de desconto 'por fora'.

Desconto racional (por dentro)

A taxa incide sobre o VALOR ATUAL (o que de fato se empresta). Fórmula do desconto: Dr = (N·i·t) / (1 + i·t). O valor atual é Ar = N / (1 + i·t).

O desconto racional é menor que o comercial para os mesmos N, i e t, pois a base de incidência (valor atual) é menor que o valor nominal. É o desconto 'matematicamente justo'.

Relação entre os dois descontos

Para os mesmos parâmetros, Dc > Dr. Existe a relação Dc = Dr·(1 + i·t), e também Dc − Dr = Dr·i·t. O desconto comercial é o racional acrescido de juros simples sobre ele.

Em provas, é comum pedir a diferença entre os dois descontos ou identificar qual regime gera maior abatimento (sempre o comercial).

Desconto composto

Existe ainda o desconto composto, em que a antecipação considera capitalização exponencial. O mais usado é o racional composto: Ar = N / (1 + i)^t, equivalente a trazer o valor nominal a valor presente por juros compostos.

No Brasil, o desconto comercial composto é pouco usado na prática; o racional composto coincide com o conceito de valor presente da matemática financeira moderna.

💡 Exemplo aplicado

Um título de valor nominal R$ 10.000,00 é descontado 2 meses antes do vencimento, à taxa de 5% ao mês (desconto simples).

Desconto comercial: Dc = N·i·t = 10000·0,05·2 = R$ 1.000,00. Valor atual: Ac = 10.000 − 1.000 = R$ 9.000,00.

Desconto racional: Dr = (N·i·t)/(1 + i·t) = (10000·0,05·2)/(1 + 0,05·2) = 1000/1,10 ≈ R$ 909,09. Valor atual: Ar = 10000/1,10 ≈ R$ 9.090,91.

Observe: o desconto comercial (R$ 1.000) é maior que o racional (R$ 909,09); o cliente paga menos no comercial, mas o banco ganha mais. A diferença é Dc − Dr = R$ 90,91 = Dr·i·t = 909,09·0,10.

🎯 Na prova

A pegadinha clássica é trocar a base: comercial incide sobre o NOMINAL, racional sobre o ATUAL. Memorize 'comercial = por fora = maior desconto'. Cuide das unidades de taxa e tempo, como em juros. Quando a questão pedir 'a diferença entre os descontos', lembre que Dc − Dr é positiva e igual a Dr·i·t. Se o enunciado não especificar, em geral assume-se desconto comercial simples (o mais cobrado).

Pontos-chave

  • Valor nominal (N) é o de vencimento; valor atual (A) é o antecipado.
  • Desconto: D = N − A.
  • Comercial (por fora): Dc = N·i·t, incide sobre o valor nominal.
  • Racional (por dentro): Dr = N·i·t/(1 + i·t), incide sobre o valor atual.
  • Para mesmos parâmetros, Dc > Dr (comercial é maior).
  • Relação: Dc = Dr·(1 + i·t).
  • Racional composto = valor presente por juros compostos: A = N/(1+i)^t.
